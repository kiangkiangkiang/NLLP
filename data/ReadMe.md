# 下載資料方式

## 資料 （臺灣司法判決書 民事資料）

1. spiritual_money資料夾為透過「**精神慰撫金**」和「**車禍**篩選內文後得到的**民事判決書**資料。
2. 資料來源官方網站：https://opendata.judicial.gov.tw/dataset?categoryTheme4Sys%5B0%5D=051&sort.publishedDate.order=desc&page=1
3. 下載時間 (爬資料的時間)：2023/2/20 ~ 2023/2/21
4. 資料所涵蓋的時間：7年，2022/11 ~ 2016/01 (2022/12尚未提供)
